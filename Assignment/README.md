# 팩토리얼 계산기 (반복 vs. 재귀 비교)
프로젝트 시작하기
이 프로젝트를 다운로드하고 실행하는 방법

1. 저장소 복제 \
git clone [저장소-URL]

2. 복제된 폴더로 이동
cd [저장소-폴더-이름]

3. 파이썬 실행방법
python 파일명.py

3. 프로그램 메뉴 및 사용법
================ Factorial Tester ================

1 반복법으로 n! 계산
2 재귀로 n! 계산
3 두 방식 모두 계산 후 결과/시간 비교
4 준비된 테스트 데이터 일괄 실행
q 종료
